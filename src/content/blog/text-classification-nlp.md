Natural Language Processing bridges human communication and machine understanding. Text classification parses unstructured strings into clean categories.

In this write-up, we evaluate text pre-processing (lemmatization, stop-word removal) and convert text into numerical matrices using TF-IDF and word vectors.

We review how classifiers interpret semantic relevance and implement a basic sentiment classifier in Python.
